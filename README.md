## Deploy on GitHub Pages

1. Commit and push this folder to your GitHub repository.
2. In GitHub, go to **Settings -> Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select your branch (usually `main`) and folder (`/root`).
5. Visit:
   - `https://<your-username>.github.io/<your-repo>/nvidia-gracs-2025/`

If you want this event page at the repository root URL, move the files from `nvidia-gracs-2025/` to the repo root before deploying.
